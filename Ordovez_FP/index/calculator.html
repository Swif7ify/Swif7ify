Hello, this is a basic calculator. Used to calculate mathematical problem. Enjoy!

Keys That can Be Pressed

v - Second Function

b - darkmode

backspace - delete recent number

delete - delete entire row

enter - calculate function

= - calculate function

ctrl + backspace = delete entire row

z - bring up notes

( and ) - insert parenthesis

sin, cos etc. - to construct a FUNCTION

Warning!

If the input have this "(" e.g log(value. Always close the expression! e.g log(value)
